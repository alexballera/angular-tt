Aprendizaje basado en proyectos

Ticmas ofrece una herramienta que permite la gestión online de proyectos interdisciplinarios mediante los siguientes pasos:

Desafío
  • Presentación del problema.
  • Planteo del problema.
  • Definición del producto final.
  • Conformación de equipos de trabajo.
Investigación
  • Análisis de la problemática a través de distintas disciplinas.
  • Comprensión interdisciplinaria del problema.
Creación
  • Prototipado del producto final.
  • Elaboración de la versión final del producto.
Presentación
  • Presentación del producto final.
  • Reflexión final sobre lo aprendido.
Habilidades del SXXI

Desde Ticmas fomentamos el enfoque de enseñanza por habilidades ya que contribuye a la formación integral de los
estudiantes. Este enfoque estimula habilidades esenciales que impulsan tanto al desarrollo personal como social.

Formación para estudiantes en plataforma:

  • Definición de cada habilidad.
  • Operacionalización de cada habilidad.
  • Ejercicios prácticos.

Guía para docentes:

  • Orientaciones para la enseñanza-aprendizaje de habilidades.
  • Dinámicas para el aula.
  • Recursos bibliográficos.
Programación

Ticmas ofrece cursos de programación de desarrollo web, dentro y fuera del aula junto con formación para los docentes.

Para secundaria: 120 hs distribuidas en 3 niveles

  • 1er año: Pensamiento computacional - Programación web desde cero - HTML - CSS.
  • 2do año: Repaso de lenguajes - JavaScript - FrontEnd - Proyecto Final.
  • 3er año: JavaScript - BackEnd - Node.js - Base de datos - Proyecto Final.

Para empleabilidad: 200 hs distribuidas 70% online y 30% presencial

  • Ingreso: pre-curso de 20 hs 100% online.
  • Fase 1: pensamiento computacional.
  • Fase 2: lenguajes de programación.
  • Fase 3: proyecto final.

Nuestro diferencial

  • Formación integral de docentes.
  • Abordaje pedagógico utilizando Habilidades del SXXI.
  • Herramienta con funcionalidad online, offline y consola embebida.
